# The annoying cookie consent
 In this cookie consent I had used the following javascript concept.
 - setTimeOut()
 - element.style
 - forms
 - formData() and .get()
 - event.preventDefault
 - CSS: row-reverse
 - toggling classes
 - 'disabled' attrribute.

Quick start:

```
$ npm install
$ npm start
````
